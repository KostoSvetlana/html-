Примечание

Главная цель проекта-создание Web-сайта. Представленная работа может быть полезна для тех, кто решил научиться создавать свои сайты.
Второстепенная цель- это практическая часть работы важна для тех, кто захочет узнать о защите природы, проблемах экологии и многое другое.
Информация представленная на сайте,взята из различных источников сети Интернет для демонстрации функционала сайта (работа вкладок,стили кнопок и т. д.)
